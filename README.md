## Install
### Easy:

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)
### Hard:

```
git clone https://github.com/Quiec/fileassistantpy
cd fileassistantpy
pip install -r requirements.txt
nano config.py // Edit Infos
python3 bot.py
```
